1. Inštalácia Django do python:
                            pip install Django
                            pip list

2. Vytvorenie Django projektu:
                            django-admin startproject nazov_projektu .

3. Nastavenie slovenčiny pre projekt (mysite/settings.py):
                                                   LANGUAGE_CODE = 'sk'
                                                   TIME_ZONE = 'Europe/Bratislava' 

4. Spustenie vývojového serveru:
                            python manage.py runserver
                            python.exe .\manage.py runserver

5. Vytvorenie aplikácie:
                    python manage.py startapp skuska

6. Zaregistrovanie aplikácie do projektu (mysite/settings.py):
                                                    INSTALLED_APPS = [
                                                    'django.contrib.admin',
                                                    'django.contrib.auth',
                                                    'django.contrib.contenttypes',
                                                    'django.contrib.sessions',
                                                    'django.contrib.messages',
                                                    'django.contrib.staticfiles',
                                                    'nazov_aplikácie',
                                                ]

7. Vytvorenie jednoduchého views (skuska/views.py):
                                            from django.shortcuts import render, HttpResponse
                                            def index(request):
                                                return HttpResponse('Hello World!')
                            
8. Vytvorenie urlpattern:
    8.1 mysite/urls.py

    from django.contrib import admin
    from django.urls import path, include

    urlpatterns = [
        path('admin/', admin.site.urls),
        path('skuska/', include('skuska.urls')),
    ]

    8.2 
        -vytvoriť súbor urls.py v priečinku aplikácie (skuska):
        -upraviť urls.py: 
                        from django.urls import path
                        from . import views

                        urlpatterns = [
                            path('', views.index, name='index'),
                        ]

11. Template:
            11.1:
                Vytvoriť v priečinku aplikácie priečinok templates/nazov_aplikácie
                V tomto pričink vytvoriť html súbor (napr. index.html)
            11.2:
                Upraviť views tak aby používal template  
                    from django.shortcuts import render
                    def index(request):
                        return render(request, 'kalkulacka/index.html')
 
12. CSS, obrázky:
                Vytvoriť v priečinoku aplikácie priečinok static/nazov_aplikácie
                V tomto priečinku vytvoriť CSS (napr. kalkulacka.css)
                Do tohot priečinku nakopírovať obrázky
                Do template (index.html) doplniť: 
                                            úplne na začiatok {% load static %}
                                            do hlavičky odkaz na style
                                                    <head>
                                                        <link rel="stylesheet" href="{% static 'kalkulacka/kalkulacka.css' %}">
                                                    </head>
                Obrázok vložiť:
                        <img src={% static 'kalkulacka/calc.png' %}>
